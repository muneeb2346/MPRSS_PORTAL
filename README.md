# MPRSS - Malware Prediction and Risk Scoring System

## Overview
MPRSS is an AI-powered Android malware detection platform that combines static analysis, dynamic sandbox analysis, and machine learning to provide comprehensive risk scoring (0-100) for APK files.

## Features
- **APK Analysis**: Upload and analyze Android applications
- **Risk Scoring**: 0-100 risk score with visual circular meter
- **Multi-dimensional Analysis**: Static, Behavioral, Anomaly, ML Confidence scores
- **Malware Classification**: Trojan, Adware, Spyware, Ransomware detection
- **XAI Insights**: SHAP-based feature importance explanation
- **Live Analysis Log**: Real-time terminal-style analysis output

## Tech Stack
